# 小程序功能与 UI 规划方案

根据您的需求，我们将把 `almond-mina` 打造为一个与 Web 端 (`almond-front`) 风格高度一致、功能互补的随身助手。核心目标是**高保真还原 Apple 设计风格**，保持**暗黑模式 (Dark Mode)** 主基调，并提供便捷的记录与提醒功能。

## 1. 整体 UI/UX 设计风格
*   **色调**: 
    *   **背景**: 纯黑 (`#000000`) 或 深灰 (`#1c1c1e` - iOS 风格卡片背景)。
    *   **文字**: 白色 (`#ffffff`) 为主，次级文字灰色 (`#8e8e93`)。
    *   **品牌色**: 青色 (`text-cyan-400` / `#22d3ee`) 和 紫色 (`text-violet-500` / `#8b5cf6`)，用于图标、按钮和强调。
*   **视觉元素**:
    *   **毛玻璃效果**: 尽可能使用透明度和模糊模拟 Glassmorphism。
    *   **圆角**: 统一使用大圆角 (如 `24rpx` 或 `32rpx`)，不仅限于按钮，卡片也保持一致。
    *   **图标**: 使用 Lucide 风格的线性图标（与 Web 端一致）。

## 2. 页面结构规划 (TabBar)
建议底部导航栏设置 3 个主要入口：

### 2.1 首页 (Home) - 快速记录与提醒
*   **顶部**: 欢迎语与头像（左上），简洁的日期显示。
*   **核心功能区**:
    *   **快速输入框**: 类似 Web 端，胶囊形输入框，支持文字录入。
    *   **类型切换**: "记知识" (Memory) / "记待办" (Task) 切换按钮 (Pill Segmented Control)。
*   **今日概览 (Dashboard)**:
    *   **消息提醒/待复习**: 显示今日需要复习的记忆卡片数量，醒目的数字角标。
    *   **待办事项**: 显示今日到期的 Top 3 任务。

### 2.2 列表 (List) - 记忆与任务管理
*   **顶部标签栏 (Tabs)**: "记忆库" | "待办清单"
*   **列表展示**:
    *   **记忆项**: 标题、下次复习时间（相对时间，如"2小时后"）、掌握度标识。
    *   **待办项**: 复选框（打钩完成）、标题、截止时间。
*   **交互**: 
    *   左滑可快速操作（如标为完成、删除）。
    *   点击进入详情页。

### 2.3 我的 (Profile) - 个人中心
*   **用户信息**: 头像、昵称。
*   **数据统计**: 累计记忆条数、已完成任务、坚持天数。
*   **设置**: 退出登录等。

## 3. 技术实现规划

### 3.1 目录结构调整
```text
apps/almond-mina/
  pages/
    tabs/
      home/       # 首页 (原 index 改造)
      list/       # 列表页 (新增)
      profile/    # 我的页 (新增)
    subpages/     # 二级页面
      detail/     # 详情页
    auth/
      login/      # 现有登录页优化
```

### 3.2 API 扩展
在 `utils/api.js` 中补充业务接口：
*   `getMemoryItems()`: 获取记忆列表
*   `createMemoryItem(content)`: 创建记忆
*   `getTasks()`: 获取任务列表
*   `createTask(content)`: 创建任务
*   `updateTaskStatus(id, status)`: 更新任务状态

## 4. 执行步骤
1.  **配置 TabBar**: 修改 `app.json`，设置底部导航和页面路径。
2.  **样式基建**: 定义全局样式变量 (`app.wxss`)，对齐 Web 端的颜色和字号。
3.  **首页开发**: 移植 Web 端的输入框组件和类型切换逻辑，实现快速添加功能。
4.  **列表页开发**: 实现记忆和任务的双列表展示，对接后端数据。
5.  **我的页开发**: 简单的个人信息展示。
